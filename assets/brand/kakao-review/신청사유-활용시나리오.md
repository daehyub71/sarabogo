## 제출용 (권장 · 약 850자)

[서비스 소개]

'살아보고(SARABOGO)'는 은퇴 시니어가 지방에서 '한 달 살아보기'를 결정하도록 돕는 웹 서비스입니다. 2026 한국관광공사 관광데이터 활용 공모전(웹·앱 구현 부문) 참가작으로, 관광공사 TourAPI와 심평원 공공데이터를 활용합니다. 관광지가 아니라 '여기서 한 달 살 수 있는가'(병원 접근성·한 달 실비·검증된 후기)에 답합니다.

서비스 URL: https://sarabogo.vercel.app

[신청 사유]

핵심 기능인 '지역 생활 인프라 지도' 구현에 카카오맵이 필요합니다.

1. 심평원이 제공하는 병원·약국 데이터는 좌표가 아닌 '주소' 기반입니다. 지도에 마커를 찍으려면 주소를 좌표로 변환해야 하며, 국내 주소 체계에 가장 정확한 카카오 로컬 API가 필수입니다.
2. 주 사용자가 60대 이상이라, 이미 익숙한 지도 UI를 제공하는 것이 학습 부담을 줄이는 가장 확실한 방법입니다.
3. '숙소에서 병원까지 차로 몇 분'은 시니어가 가장 먼저 묻는 질문이며, 이 지표가 서비스의 핵심 차별점입니다.

[활용 시나리오]

- 지역 목록 화면: 카드에 '병원 차로 8분 · 한 달 82만원' 등 생활 지표를 표시합니다. (구현 완료)
- 지역 상세 지도 화면(카카오맵 적용 예정): 카카오맵 JavaScript SDK로 지도를 렌더링하고 병원(적색)·약국(녹색)·마트(주황)·관광지(청색)를 종류별 마커로 표시합니다. 병원·약국은 심평원 API로 수집해 카카오 로컬 API로 좌표 변환하며, 관광지·숙소는 TourAPI로 수집합니다. 마커 클릭 시 시설명·주소 정보창을 노출하고, 숙소 기준 병원까지의 이동 시간을 계산해 보여줍니다.
- 맞춤 추천 결과 화면: 추천된 한 달 코스의 방문지를 지도에 표시해 동선을 확인하게 합니다.

Next.js(PWA)로 개발 중이며 TourAPI 연동과 지역 카드 화면을 구현해 위 URL에 배포했습니다. 카카오맵 이용 정책(지도 위 표시·출처 표기)을 준수하겠습니다.

## 더 짧게 잘릴 경우 (최소 버전)

[서비스 소개]
'살아보고'는 은퇴 시니어의 '한 달 살아보기'를 돕는 웹 서비스입니다(2026 한국관광공사 관광데이터 활용 공모전 참가작). 관광지가 아니라 '여기서 한 달 살 수 있는가'에 답합니다. URL: https://sarabogo.vercel.app

[신청 사유]
'지역 생활 인프라 지도' 구현에 카카오맵이 필요합니다. 심평원 병원·약국 데이터가 좌표가 아닌 '주소' 기반이라 카카오 로컬 API로 좌표 변환이 필수이며, 주 사용자가 60대 이상이라 가장 익숙한 지도 UI가 필요합니다. '숙소에서 병원까지 차로 몇 분'이 서비스의 핵심 지표입니다.

[활용 시나리오]
카카오맵 JavaScript SDK로 지역 지도를 렌더링하고 병원·약국·마트·관광지를 종류별 마커로 표시합니다. 병원·약국은 심평원 API 수집 후 카카오 로컬 API로 좌표 변환하며, 관광지는 관광공사 TourAPI로 수집합니다. 마커 클릭 시 시설명·주소를 노출하고, 숙소 기준 병원까지 이동 시간을 표시합니다. 지역 카드 화면은 구현 완료해 위 URL에 배포했습니다. 카카오맵 이용 정책을 준수하겠습니다.

## 함께 확인할 것 (반려 방지)

- **서비스 정보 URL**: `https://sarabogo.vercel.app` — 익명 접근 200 확인 완료.
  ⚠️ `vercel ls`가 보여주는 긴 배포 URL(`sarabogo-xxxx-...vercel.app`)은 **SSO 보호로 302 리다이렉트**되어
  심사자가 못 본다. 반드시 짧은 별칭을 쓸 것.
- **도메인 등록 2곳** (안 하면 배포판에서 지도가 안 뜬다):
  - 앱 설정 → 플랫폼 → Web 플랫폼
  - 앱 설정 → 앱 → 플랫폼 키 → JavaScript SDK 도메인
- 첨부 이미지에 개인정보 없음 (데모 데이터만 사용).
- 지도 화면은 미구현이므로 첨부 이미지에 "구현 예정(기획안)"으로 명시했다.
